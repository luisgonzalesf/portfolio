# 📊 Proyecto: Análisis de Productividad Agrícola

## 🎯 Objetivo del Proyecto
Construir un pipeline end-to-end que integre múltiples fuentes de datos y genere dashboards ejecutivos para seguimiento de campañas agrícolas.

## 🧠 Insights Encontrados

- El Fundo Norte tiene 18% más rendimiento que el promedio.
- La campaña 2024-2 mostró menor margen por aumento de costos.
- Existe correlación entre fertilización y productividad.
